# Install Dependendencies

## Node Package Manager

## Firebase CLI
```
npm install -g firebase-tools
```

## Sign into Firebase to gain access to projects
```
firebase login
```
## Create React App
```
px create-react-app [app name]
```
## Install React Router DOM
```
npm i -D react-router-dom
```
